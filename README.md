# DrowsinessDetectionIOT
# Check app deployed at https://drow-deploy.herokuapp.com/

## Dependencies

1. Python 3
2. opencv
3. dlib	
4. imutils 
5. scipy
6. numpy
7. argparse

## Run 

```
Python3 drowsiness_yawn.py -- webcam 1		
```
